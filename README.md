#Sales Analysis Dashboard 📊
Overview

The Sales Analysis Dashboard is an interactive Power BI dashboard designed to analyze sales performance across different years, regions, categories, and customers. It helps businesses track revenue trends, customer behavior, and order insights for better decision-making.

This dashboard provides a bird’s-eye view of business health, like a control tower watching the whole marketplace river flow 🚢.

Features

Key Performance Indicators (KPIs):

Total Sales
Total Customers
Total Orders
Average Sales per Order

Visual Insights:

Sales by Year → Tracks yearly growth trends
Sales by Region → Compares regional performance
Sales by Category → Shows category-wise contribution
Sales by Customer → Identifies top customers

Interactive Filters:

Region filter
Category filter
Date range slicer

Tech Stack
Power BI for dashboard creation
Excel / CSV for dataset
DAX (Data Analysis Expressions) for calculations
Power Query for data cleaning and transformation
Dataset Information


Insights from Dashboard

📌 Sales show steady growth from 2016 to 2018

📌 West region generated the highest sales

📌 Technology category has the maximum contribution

📌 Top customers contribute significantly to overall revenue

📌 Seasonal and regional trends can be analyzed using date slicers
